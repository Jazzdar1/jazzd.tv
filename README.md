# IPTV

Watch 500+ live india TV channels with IPTV **Android Application**

[**Download Latest APK From Release**](https://github.com/kananinirav/iptv-indian-app/releases/)

## Feature

- 500+ live Tv Channels
- Search Channels
- Full Screen View
